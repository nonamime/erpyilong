# 赤龙ERP

旨在为小微企业提供一款**免费**、**开源**、**满足业务**、**灵活稳定**的ERP管理系统

## 主要功能

- 采购/销售订单管理
- 付款/收款单管理
- 凭证管理
- 主数据管理
- 用户与权限
- 职员与部门
- 数据字典
- 单点登录
